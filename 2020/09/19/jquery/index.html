jQuery是什么？

是封装了常用js操作函数的一个库文件

jQuery=javascript+Query 即 强大的DOM节点查询

下载

从官方网站下载时，jquery的文件名格式如下：

jquery-1.x.x-min.js 或 jquery-1.x.x.js

官方网站：http://jquery.com

官方手册：http://api.jquery.com

选择器

基本选择器

选择器：id $('#test1').css('background','gray');

​ 标签$('p').css('background','gray');

​ 类$('.test1').css('background','gray');

​ 群组$('#test1','.test1','p');

层次选择器：

​ $('#test p') 空格代表祖先和后代的关系

​ $('#test > p')父子关系

​ $('#test +p')下一个弟弟，挨着的元素

​ $('#test ~ p')所有弟弟p标签

属性选择器：

​ $("input[name='email']").css('background','gray');

过滤器

如果选择器选出的结果仍不够细，可以用过滤器，进一步筛选， 过滤器一般写在”选择器”之后，如”选择器:过滤器”;

基础过滤器

是根据对象的索引值来查找

$('li:first').css('background','red'); //  找第 1 个
$('li:odd').css('background','blue'); //  找第奇数个 , 从 0 计数
$('li:eq(2)').css('background' , 'purple'); //  找第 2 个 , 从 0 计数

内容过滤器

​ $(‘tr:contains(字符)’) 内容过滤
​ $(‘div:has(h1)’) 含有h1标签
​ $(‘a[href]’) 找含有href属性的a标签

操作属性

普通属性
$('img').attr('src','on.jpg'); 
css属性
$('p').css("background","red");
布尔属性：
$('input:checkbox').prop('checked',true);

节点操作

 删除DOM节点
$('li:last').remove();
 清空内容 
$('li:last').empty();
  增加节点
$('ul').after($('<ol><li>哈哈</li></ol>'));
$('ol').append($('ul li:last'));
  节点包裹
$('input:text').wrap('<p></p>');//input标签分别包裹在<p>标签里
$('input:text').wrapAll('<p></p>');// 所有inpuy标签包裹在一个<p>标签里
$('li').wrapInner('<b></b>');//包裹文本不包括节点
jquery与DOM对象的关系和转换

$()返回的是什么？是jquery对象不是dom对象。dom对象装在了jquery对象里索引分别是0，1，2….
jquery对象转为DOM对象
var li=$(‘li’)[0];//得到第一个li
DOM对象转为jquery对象
var li=document.getElementsByTagName(‘li’)[3];
$(li).css(‘background’,’gray’);

事件

原生事件和jquery事件除了语法上稍显区别，最重要的是绑定次数：

原生多次绑定事件会被最后一次覆盖前面的事件。而jquery绑定事件都会执行。

独特的ready事件

相当于原生的onload属性，但ready不同。onload是在加载完毕后执行，ready是一开始就先执行
$(document).ready(function(){});简写：$(function(){})

jquery特有的绑定函数：

​ bing() unbing()
​ //绑定click事件
​ $(‘#test’).bind(‘click’,function(){});
​ //取消绑定click事件
​ $(‘#test’).unbind(‘click’);
​ //绑定一次事件，执行一次
​ $(‘#test’).one(‘click’,function(){});
​ //动态绑定，为后来增加的节点也绑定
​ $(‘.test’).live(‘click’,function(){});

事件委托

​ 给父元素绑定事件，子元素都能执行
​ 使用 on()函数
​ $(ul).on('click','li',function(){});

事件对象

$('div').click(function(ev){ alert(ev.type + ':' +ev.which); });
